# POE Portfolio. 

A modern, single‑page portfolio website built with vanilla JavaScript and Tailwind CSS. 
It showcases projects, technical skills, and includes a contact form – all in one HTML file. 
Designed for quick deployment and easy customisation.

## Tech used
> HTML5
> Tailwind CSS
> JavaScript (Vanilla ES6)
> Google Fonts

>> No server needed – it’s a static site.

1. Download or clone the repository.
2. Open `index.html` in your browser.

That’s it – everything is self‑contained.
